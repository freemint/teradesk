Русский файл ресурсов и гипертекстовый документ для TeraDesk версии 4.08
(http://solair.eunet.rs/~vdjole/teradesk.htm).

Архив содержит:
----------------
-DESKTOP.RSC : Файл ресурсов
-DESKTOP.RSD : Определения файла ресурсов (в формате Interface)
-TERADESK.HYP : Гипертекстовый файл в формате ST-Guide
-TERADESK.REF : Файл ссылок для .HYP в формате ST-Guide
-README.TXT   : Этот файл


Установка:
------------------
Переименуйте оригинальный файл DESKTOP.RSC в .RSX и скопируйте новый русский файл ресурсов в основную папку программы.

Скопируйте TERADESK.HYP и TERADESK.REF в основную папку для гипертекстовых документов в формате ST-Guide (например в C:\Guides).
